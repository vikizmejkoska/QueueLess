# QueueLess

QueueLess is a mobile-first smart queue and appointment management app for beauty salons.

## Features
- Browse salons and services
- Take a virtual ticket
- Track ticket status and estimated wait time
- Admin login and queue management
- Dashboard for active queue overview

## Tech Stack
### Backend
- Node.js
- Express.js
- PostgreSQL
- Prisma
- JWT Authentication

### Mobile
- React Native
- Expo
- Zustand
- React Navigation

## Project Structure
- `backend/` - REST API, database logic, authentication, queue management
- `mobile/` - React Native mobile app for clients and admins
